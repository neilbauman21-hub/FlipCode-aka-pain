spec

  • flip N - toggle transistor N (0-1048575)
  • clock N - read entire transistor state and output as bytes
  • end - halt execution
  • # comment - ignored line
  • behavior: like real computer memory - reads all transistors from T0 to highest active
  • output: every 8 transistors = 1 byte (T0-7, T8-15, T16-23, etc)
  • storage: sparse map - only active transistors consume memory
